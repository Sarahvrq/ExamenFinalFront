# Examen Final — Next.js

Aplicación web desarrollada con Next.js y TypeScript que consume la API pública de Rick & Morty para mostrar characters, y hacer diferentes queries mediantes filtrajes. Adicionalmente, puedes ver informacion de episodios y ubicaciones de la serie.

## Instalación y Configuración

Para ejecutar este proyecto localmente, sigue estos pasos:

1. **Instalar dependencias:**

   ```bash
   npm install

   ```

2. **Arrancar el proyecto:**

   ```bash
   npm run dev

   ```
